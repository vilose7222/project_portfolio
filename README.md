# 미리보기
![ezgif com-gif-maker](https://github.com/vilose7222/final_project_go90/assets/141006779/38316584-546c-4dea-91e5-60a6dc2eed59)



## 개요

기존 HTML 로 만든 포트폴리오에 DB연동과 JSP를 활용한 프로젝트 

## 주요 기능 

1.회원가입, 로그인 , 로그아웃

2.게시판 글쓰기

3.사진과 설명글 DB관리

## 개발 인원

개인프로젝트
