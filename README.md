## 개요

기존 HTML 로 만든 포트폴리오에 DB연동과 JSP를 활용한 프로젝트 

## 주요 기능 

1.회원가입, 로그인 , 로그아웃

2.게시판 글쓰기

3.사진과 설명글 DB관리
